El juego es sencillo y se desarrolla de la misma forma que Pokemon de Nintendo, aunque con sus diferencias obvias.

Pimero se debe elegir entre los tres pokemon principales (la verdad me hubiese gustado poner más, pero tuve inconvenientes con encontrar imagenes con los pokemon de espalda y no me fue nada bien con la api de pokemon, en algún momento con más tiempo desarrollare mas el juego)
Despúes elegis entre tres entrenadores que representan la dificultad del juego, cada entrenador cuenta con 3 pokemon para la batalla.
Una vez elegido todo se pasa a la pantalla de batalla, donde dependiendo el nivel aparecen distintos pokemon.

El ataque del rival se define por la función Math.random, por lo tanto durante la pelea el rival hace distintas cosas.

También en el menu hay un boton con curaciones para aumentar la vida, el rival también cuenta con esa función.

Cada acción se describe en el cuadro inferior izquierdo, ahí aparece que hace cada pokemon y entrenador.
